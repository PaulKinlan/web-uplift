# web-audit

Audit a URL for modern web UX quality (explore -> test plan -> principles +
Modern Web Guidance -> findings -> prioritised task list; --fix to apply
guidance-backed fixes and re-audit).

Read the file `.claude/skills/web-audit/SKILL.md` and follow its instructions
exactly. Treat the text the user typed after the command as the arguments (the
URL, plus optional flags like `--plan-only`, `--path <id>`, `--out <dir>`,
`--replan`, `--fix`, `--source <dir>`).
